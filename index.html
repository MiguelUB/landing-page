Guia simple para usar animated.css y bootstrap

En ambos casos lo mas importante es agregar las lineas en el head
link rel = "stylesheet" href = "/node_modules/animate.css/animate.css"
link rel="stylesheet" href="/node_modules/bootstrap/dist/css/bootstrap.css"
Esto usara el codigo que se instalo con npm install en node_modules y como son archivos css solo basta con impotarlos ambos se comportan de manera similar con solo agregar una clase a algun tag ya se aplicara el estilo de css personalizado
